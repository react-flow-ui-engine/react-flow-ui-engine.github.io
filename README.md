# react-flow-ui-engine docs website

A standalone documentation website for `react-flow-ui-engine`.

## Open locally

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
npx serve .
```

## Deploy

This folder is static HTML, CSS, and JavaScript. You can deploy it to GitHub Pages, Vercel, Netlify, Cloudflare Pages, S3, or any static hosting service.

## Included sections

- Why this library exists
- Installation and quick start
- Core concepts
- Flow definition API
- Conditional navigation
- Conditional same-page content
- React component content blocks
- React/core API reference
- Examples and local development
